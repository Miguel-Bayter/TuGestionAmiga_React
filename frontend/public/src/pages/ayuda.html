Ayuda

En esta sección están las dudas más comunes y una guía rápida de uso.

Preguntas frecuentes

No aparecen libros, ¿qué hago?
Verifica que el servidor esté corriendo en http://localhost:3000 y que tu tabla libro tenga datos. Para cargar datos de ejemplo, ejecuta los scripts en backend/sql (por ejemplo seed_demo_data.sql o seed_admin_and_books.sql) y recarga la página.
Me sale “Cannot POST /api/...”, ¿por qué?
Eso pasa cuando abres el HTML con file:// o con un servidor distinto (por ejemplo Live Server en otro puerto). Para que los endpoints /api/* funcionen, abre las páginas desde el backend Node: http://localhost:3000/index.html.
¿Cómo rento y devuelvo un libro?
Primero inicia sesión. Luego, desde el listado selecciona Rentar. Después entra a Préstamos y usa el botón Devolver. Cuando devuelves, el backend vuelve a marcar el libro como disponible.
¿Cómo compro un libro?
Hay dos interfaces:
  • Frontend HTML (frontend/public): el botón Comprar registra directamente la compra (POST /api/compras).
  • Frontend React (frontend/react): Comprar agrega al carrito (POST /api/carrito) y luego se confirma en Carrito con el checkout (POST /api/carrito/checkout).
¿Cómo restablezco mi contraseña?
En Login, usa ¿Olvidaste tu contraseña?: primero generas un código y luego lo usas para crear una nueva contraseña. En esta maqueta el código aparece como demo_code para poder probarlo sin correo.

Guía rápida

  1. Configura la BD y el .env.
  2. Arranca el servidor desde backend/ con npm start.
  3. Abre http://localhost:3000/index.html.
  4. Inicia sesión y prueba rentar/comprar.
